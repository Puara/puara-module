# Foreword

Welcome to the `puara-module` documentation!

`puara-module` is a library for the ESP32 that provides a simple interface for managing WiFi, a webserver, mDNS, and a filesystem. It is designed to be used with the Arduino framework.

This book will guide you through the process of using `puara-module` in your projects. You will learn how to:

*   Compile the library and examples.
*   Use the `Puara` object to manage your device.
*   Configure your device using JSON files.
*   Use the integrated webserver.
*   Manage WiFi connections.
*   Work with the filesystem.
*   Communicate over serial.
*   Use the mDNS service.
*   Understand the provided examples.
